# Comma Separated Values

Programa que muestra en una tabla por filas y columnas dígitos o palabras entrecomilladas separadas por comas.

## Uso

Escribir las palabras (o dígitos) que se quiera que se muestren en la tabla, entrecomilladas y separadas por comas. Si pasas a la fila siguiente, pones un retorno de carro. Un ejemplo:

"pokemon", "tipo"  
"pichu", "electrico"  
"bulbasaur", "planta"  


OJO: No poner un retorno de carro después del último elemento o dará error.
